Another Place

A Web AR Experiment By Dylan Dougherty

What

Another place is a mobile experiment built for the mobile web browser. It displays a custom 3D model ontop of a defined AR marker inside of the phone's web browers by using the camera and various sensors built into the device.

How

This experiment runs on AR.JS, a powerful and light-weight augmented reality library for the web. AR.JS utilizes the motion tracking capabilities of Aframe.io, and the 3D rendering of Three.js. It then adds additional marker tracking that utilizes the device's camera to display content ontop of markers in the real world. The model for this experiment was created and animated in Blender and then exported as an .glb file for use in AR.JS. All source files can be found in the Github Repo and are available for free under the MIT license.
