Tables Games

An AR Experiment By Dylan Dougherty

What

Table Games is an experimental augmented reality pong game meant to be played on a custom-built table. Table games utilizes computer vision and tracking techniques to control paddles which two players can use to play the game.

How

The Table Games application runs on Processing 3.5.3 and uses the Reactivision computer vision framework. A projector and camera are placed underneath a custom-built table with a plexiglass top and difusion layer. The Projector displays the Table Games processing application on the plexiglass, while the camera tracks Fiducial markers placed on the underside of two "controllers." Each player can slide their controller across the tabletop to try and hit a ball back towards the other player. If a player misses, a goal is made and the score increased by one. Once a player reaches a score of 10, a winner is declared and the game ends. All source files can be found in the Github Repo
